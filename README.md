## 🚀 Featured Projects

*May 2025 – March 2026*

- Architected AWS infrastructure — VPC for network isolation, least-privilege IAM roles, EC2 for hosting, S3 for model weights and data
- Containerized backend (FastAPI) and frontend (React) with Docker, using `docker-compose` to run both services together
- Built and automated a CI/CD pipeline with GitHub Actions to test, build, and deploy on every push, using smoke-test scripts to keep CI fast
- Two-stage ML pipeline in production: CNN segmentation model (**93.99% Dice Coefficient**) + Random Forest classifier (**92.90% F1-score**)
- Validated the complete workflow end-to-end — image upload → inference → response — across local and deployed environments

`AWS` `Docker` `GitHub Actions` `FastAPI` `React` `CNN` `Random Forest`

- Trained a Random Forest model to predict crop yield, with a full data cleaning and preprocessing pipeline
